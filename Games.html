SVAROG

A 2D pixel-art game about slavic mythology. This is the main game I'm working on. The project started back in, just when I recently learned the basics of Unity Game Engine. I'm solo developing this game so everything you see has been made by me, except some things like the music which you can find credited on the game's page. It's still early in development and it will be fully released on Itch.io and Steam. Svarog won the 1st place in local high school competition in 2024. Rest of the information about the game is on it's Itch.io page.

Itch.io: https://bloodassassin89.itch.io/svarog




Paper Sharp

A game about math duels in 17th century Venice. I've started this game as high school project with a couple of my friends, Luka Pavičević and Stefan Jovčić, we won 1st place in local high school competition in 2023 as well as 2nd place in Neumann IT competition in Hungary. I am the project lead, all of the art and most of the code has been done by me.

Itch.io: https://bloodassassin89.itch.io/paper-sharp

Dungeon Guardian

This is the first game I've started working on. It's a 3D roblox game inspired by The Witcher 3 and Hades. It was a big bite to swallow and that's why I dropped it after a year, besides that Roblox Studio was working against my vision.

I've learned a lot from this project which lead me to making Svarog.
